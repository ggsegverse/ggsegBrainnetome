# ggsegBrainnetome: Brainnetome Atlas for the 'ggseg' Ecosystem

Brainnetome atlas for the 'ggseg' ecosystem. Provides a 'ggseg_atlas'
object with 2D polygon geometry for use with 'ggseg'.
